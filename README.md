# Customer Ticket Classification (NLP)

This project demonstrates a machine learning system for classifying customer support tickets using natural language processing.

## Features

- TF-IDF text vectorization
- Two ML models
  - Logistic Regression
  - Support Vector Machine (SVM)
- Adjustable confidence threshold
- Input validation for noisy text
- Top-5 prediction scores
- Streamlit web interface

## Tech Stack

- Python
- Scikit-learn
- TF-IDF
- Streamlit
- Joblib

## Demo

Users can input a customer support message and the system will:

1. Predict the ticket category
2. Display model confidence
3. Show top classification scores

## Deployment

The application can be deployed using **Streamlit Cloud**.
